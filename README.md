# 🧱 BuildBox Demo

Welcome to **BuildBox Demo**!  
A block-based building and exploration prototype where you can test mechanics, interact with blocks, and explore a simple sandbox world.

---

## 🎮 About the game

**BuildBox Demo** is a prototype focused on basic 3D building mechanics.  
Explore the world, break and place blocks, and experiment with the system freely.

---

## 🕹️ Controls

- **W / S** → move camera  
- **Left Mouse Button** → break blocks  
- **Right Mouse Button** → place blocks  
- **F** → open inventory  

---

## ⚙️ Technologies used

- HTML5  
- JavaScript  
- CSS
